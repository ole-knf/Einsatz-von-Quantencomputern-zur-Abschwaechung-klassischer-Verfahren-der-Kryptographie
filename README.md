# Einsatz von Quantencomputern zur Abschwächung klassischer Verfahren der Kryptographie

Dieses Repository fasst alle Codeabschnitte aus meiner Bachelorarbeit zum Thema "Einsatz von Quantencomputern zur Abschwächung klassischer Verfahren der Kryptographie" zusammen. Die Implemntationsbeispiele mit Qiskit basieren auf den Beispielen von Qiskits Dokumentation. In den jewiligen Dateien ist der Ursprung des Codes verlinkt. Etwaige Erklärungen zu Codeabschnitten sind der Bachelorarbeit zu entnehmen.
